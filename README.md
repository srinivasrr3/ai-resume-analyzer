\# AI Resume Analyzer



AI Resume Analyzer is a full-stack web application that evaluates a resume against a given job description and generates an ATS (Applicant Tracking System) compatibility score. The system identifies matched skills, missing skills, and provides improvement suggestions to increase the chances of resume shortlisting.



The application is built using Angular (Frontend) and Spring Boot (Backend) and includes modern UI animations and intelligent resume parsing.

\## Running the Backend



Navigate to backend folder



cd resume-analyzer-backend



Run the Spring Boot server



mvn spring-boot:run



Backend runs on



http://localhost:8080



---



\## Running the Frontend



Navigate to frontend folder



cd resume-analyzer-frontend



Install dependencies



npm install



Run Angular server



ng serve



Frontend runs on



http://localhost:4200



---



\## How the System Works



1\. User uploads resume

2\. User enters job description

3\. Backend extracts resume text

4\. Skills are detected from resume

5\. Resume skills are compared with job description

6\. Matching and missing skills are identified

7\. ATS score is calculated

8\. Results are returned to frontend

9\. UI displays results with animations and suggestions
